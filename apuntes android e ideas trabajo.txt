En mainActivity hay un onClick. En el layout (activity_main.xml) hay algunos componentes con onClick

Dice mínimo 3 actividades, pero para no hacer tantas actividades se puede usar fragmentos (varias funcionalidades en una misma actividad

Proyecto, new, activity, Basic Views Activity (es el que crea los fragmentos) / Empty Views Activity (vacío, sin fragmentos). Eso crea la clase java, el layout y lo registra en el manifest. Nunca se hace un new Activity class a pelo. Al eliminar, lo mismo: hay una opción para eliminar del manifest, etc.
